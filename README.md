## 🚀 Featured Projects

### 🗺️ [Waze Clone — Real-Time GPS Navigation & Traffic System](https://github.com/elir0n)
> **C · Python · Docker · OpenStreetMap · Leaflet.js · pthreads**

A real-time GPS navigation engine modeled after Waze, built on Tel Aviv's actual road network (~6,500 nodes, ~13,000 edges from OpenStreetMap).

- Architected a **3-tier concurrent TCP server** in C: accept thread → per-client threads → routing/traffic worker pools with `pthread_rwlock_t`
- Implemented **A\* pathfinding** with a binary min-heap and EMA-smoothed live edge weights for accurate routing under traffic
- Simulated **30,000 vehicles** with congestion physics — speed degrades proportionally to road occupancy, modeled by road class and lane count
- Built a full-stack web UI (Python HTTP↔TCP bridge + Leaflet.js) with live car positions, congestion overlays, and interactive route planning
- Deployed the complete stack with **Docker Compose**
